Algorithm

The Wordleist solve function starts by testing a fake 'word' comprising the five most frequent letters in the candidate word list. It inspects the pattern returned from the wordle function, and elimininates all those words which are not consistent with that pattern.

It then takes the first word remaining in the candidate list, and repeats the cycle, until

  1. A pattern matches all five characters;
  2. No word remains in the candidate set;
  3. All six turns have been exhausted.

Evaluation

Wordleist currently solves 77% of puzzles within six attempts, but not all. A more sophisticated generate function would improve this performance, but it's amazing how successful the 'just test the first remaining of the non-eliminiated candidates' algorithm is.
