pattern?

(pattern? pattern)

Validate wordle patterns. Return true if pattern is a valid pattern, else false.

A pattern is a sequence of word-length sequences, such that: 1. every element is a sequence, or nil; 2. the first element of every sequence which is not nil is a keyword; 3. the second element of every sequence which is not nil is a character.

solve

(solve)(solve game)(solve game patterns)(solve game patterns candidates)

Solve a wordle; with no arguments, start with a new random word; otherwise, arguments are: * game a function of one argument, as generated by wordle-gen, which takes a guessed word as argument and returns a pattern; * patterns a list of patterns from prior attempts; * candidates a list of words to use as candidates for guessing.
